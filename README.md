## 📊 Data Structure

Each trade entry includes:
- Date, Time, Symbol, Timeframe
- Direction (Long/Short) & Leverage
- Entry/Exit/Stop-Loss Prices
- Take-Profit Levels
- Risk & Reward Percentages
- RR Ratio (Risk/Reward)
- Detailed Technical Analysis Notes
- Chart Screenshots (base64)
- Pre-trade Emotion & Rules Followed

## 🔧 Analysis Tools

- **`analysis.ipynb`**: Jupyter notebook that loads the JSON and computes all metrics
- **`trading_performance_dashboard.png`**: Visual dashboard showing P&L, drawdown, and distribution charts
- **`trading_summary_advanced.csv`**: Exported summary table
